## Usage

1. Clone the repo.

    ```
    git clone https://github.com/junsircoding/js_game_2048.git
    ```

2. Start a simple server with Python 3.

    ```
    python -m http.server
    ```

3. Open a browser and access `localhost:8000` .

Enjoy it!
